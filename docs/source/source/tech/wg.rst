wg.py
==========
There are two different waveguides in IMECAS SiN pdk, which are defined as follows:

#. ``Channel``
    #. ``Channel.C``

        * COR_WIDTH = 1.5 um
        * CLD_WIDTH = 21.5 um
        * TCH_WIDTH = 10 um

    #. ``Channel.O``

        * COR_WIDTH = 1.1 um
        * CLD_WIDTH = 21.1 um
        * TCH_WIDTH = 10 um
